# Backend requisitos

## java 11, maven

### Para executar
" cd back-end/pet-tinder && ./mvnw spring-boot:run"

### Swagger
" http://localhost:8080/swagger-ui.html" 



# Frontend requisitos

## Node 14 e NPM

### Para instalar
" cd front-end/ && yarn"

### Para executar
" yarn start"

Na sequência será aberta uma página em http://localhost:3000
